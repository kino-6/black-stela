# Advanced Vocation Source Masters

P21 stores the gendered advanced-vocation base masters here before any runtime
derivative is exported.

Filename:

`vocation-<slug>-human-<male|female>-base.png`

Every master is 1024×1536 PNG RGBA with clean alpha, neutral lighting, no
floor, scenery, particles, or baked shadow. Male and female masters are equally
practical and authoritative; they are separate people, never recolors.

These are **not** selected from a class or a name. Runtime wiring waits for an
adventurer's stored appearance lineage, so an advanced vocation cannot turn an
existing character into another face or gender. A supplied `visualProfile`
always remains authoritative.
