# Verdant Advanced Vocation Source Masters

P21 stores each living-grove advanced-vocation base master here before any
runtime derivative is exported.

Filename:

`vocation-verdant-<slug>-human-<male|female>-base.png`

Every master is a 1024×1536 PNG RGBA full figure with clean alpha, neutral
lighting, and no scenery, floor, particles, or baked shadow. Male and female
masters are distinct people with equally practical role equipment, never a
recolor.

These files do not select themselves from a vocation, name, or party slot.
Runtime integration waits for a stored appearance lineage, so changing vocation
cannot replace the registered person. A supplied `visualProfile` remains
authoritative.
