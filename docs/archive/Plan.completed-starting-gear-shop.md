### [x] Lane W: Starting Gear, Categorized Shop, and Pre-Purchase Preview

Goal: make equipment feel like a DRPG preparation layer from registration
through shopping, not a flat item list.

Planned slice:

- [ ] Define class-appropriate starting loadouts: at minimum a weapon and basic
  protection for front-line jobs, a usable tool/weapon for rear-line jobs, and
  no empty equipment slots unless the class fantasy or scenario explicitly
  calls for it.
- [ ] Show starting gear during adventurer registration and suggested recruit
  confirmation so the player understands what the character can actually do.
- [ ] Treat starting gear as real inventory/equipment, so shop comparison has a
  current item to compare against.
- [ ] Split shop stock into controller-friendly categories: weapons, armor,
  shields/offhand, head/hands/accessories, tools, and consumables.
- [ ] Keep category switching stable and readable in Japanese and English,
  without moving command positions when messages update.
- [ ] On highlighted equipment, show whether the selected adventurer can equip
  it before purchase.
- [ ] If unequippable, show a clear warning and block accidental purchase or
  require an explicit confirmation path.
- [ ] If equippable, compare against the currently equipped item in that slot:
  attack range, defense, accuracy, speed, special tags, price, and remaining
  gold after purchase.
- [ ] Support party-member switching so the player can compare the same item
  against each adventurer before buying.
- [ ] Add Browser Self-Play or targeted Playwright evidence for category
  traversal, unequippable warning, equippable comparison, and Japanese labels.
